## Project Overview

This project is a responsive hospital website designed and developed for Galaxy Hospital Ltd to improve their online presence and accessibility for patients.

The website focuses on clear information delivery, professional healthcare UI, and easy navigation for visitors.

This project was completed as part of my academic learning and freelance/client work experience.

## Features

- Responsive design (desktop & mobile friendly)
- Multi-page layout (Home, About, Services, Doctors, Testimonials, Contact)
- Clean and professional healthcare UI
- Easy navigation and readable content
- Optimized images and structured layout
